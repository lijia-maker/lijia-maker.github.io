如何推送本地项目到github

在github上面创建一个项目地址

本地仓库操作

进入到本地项目的目录文件下面,打开命名行

初始化本地仓库

git init

关联远程仓库

git remote add origin 远程仓库地址

添加全部已经修改的文件

git add -A

将修改后的文件提交到本地仓库

git commit -m ‘注释’

同步

git push -u origin master第一次需要加-u(main是指分支)

第二次不用加
git push origin main
